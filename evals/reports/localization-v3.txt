CORRECTED FINDING (replaces the old 'neural model cannot beat the rule, so it is not built'):
1. The oper-state RULE is PROVABLY BLIND to gray faults (AUROC exactly 0.500) — under load the
   link stays oper-up, so state alone carries no signal. It is strong on blunt faults (~0.92).
2. LEARNED MODELS RECOVER GRAY-FAULT LOCALIZATION THE RULE CANNOT — even from frozen v1
   aggregate features (MLP ~0.67, GNN ~0.69-0.72). The old conclusion was a hollow-corpus artifact.
3. THE GRAY SIGNAL LIVES SUBSTANTIALLY IN TEMPORAL STRUCTURE: adding strictly-causal temporal
   channels (v1->v2) lifts gray AUROC to ~0.91 and gray top1 from ~0.35 to 0.73-0.89 (MLP). The
   controlled ablation localizes WHICH feature class carries the signal.
4. MODEL COMPARISON IS NUANCED (reported, not hidden): the GNN's relational message-passing leads
   under impoverished v1 features (impaired 0.721 vs MLP 0.664), but temporal features let a
   simpler MLP match/beat it (impaired v2: MLP 0.910 vs GNN 0.721, flat) — a data/tuning limit on
   only 26 training incidents, stated plainly.
5. HONEST CEILING & CONTROL: gray top1 under aggregate features ~0.35; healthy_control at chance
   for all methods (no manufactured localization, no position leakage).
